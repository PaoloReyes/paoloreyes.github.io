What is it about?

RLC (Room Light Changer) is an IoT device that allows remote control of the lighting in a room or keeps track of the number of people inside the room and automatically turns off the lights when it is empty.


The device is an evolution of a previous project called Room Occupancy Counter, which aimed to limit the maximum number of people in a room with the onset of the pandemic. Using the same concept, I maintained the original functionality but changed the brain of the system from using an Arduino Nano to using an ESP32, which enabled WiFi connectivity. This allowed me to create a smartphone application to turn the lights in my room on or off with just one click.


This device had many other features in addition to the ones already mentioned. Some of these features include:

  • Humidity Meter.
  • Temperature Meter.
  • Local Control Interface with Buttons.
  • Local Data Display with LCD Screen.
  • Alarm to indicate overcrowding.
  • Wi-Fi connectivity.
  • Bluetooth connectivity.
