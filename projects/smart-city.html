What is it about?

This was a school project with the idea of ​​using microphones and signal processing techniques to identify an ambulance at each intersection in a city and prioritize traffic lights accordingly to minimize crossing time and save lives using the sound of its siren.


For this project I used a correlation algorithm running on a real time operating system (FREERTOS) in 4 different esp32 each one processing 2 independent microphones, one remote communication via mqtt and the output sequence of the traffic lights in a multicore approach using 4 threads one for each task. Aditionally all the data is sent to a "Monitoring Office" (my laptop) to visualize the ambulance path, statistics about its time to cross and the correlation coefficients on each microphone.
