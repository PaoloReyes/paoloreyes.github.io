3D Manipulation

This project was intended to learn how Kalman filters work. The idea was to mix two independent sensor data from an accelerometer and a gyroscope to remove vibrations produced when you only use the accelerometer and drifting when you only use the gyroscope.


The visualization is done by sending the retrieved roll, pitch, and yaw angles from the sensors after applying the Kalman filter to the roll and pitch angles to my laptop using the Processing programming language. This allows the use of its 3D visualization interface and reference frame rotations to visualize the object orientation in a way that is more intuitive than a tuple of three numbers by projecting a 3D fixed object in its original reference frame into a rotated reference frame with the corresponding RPY values.
