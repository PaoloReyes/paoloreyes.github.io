What is it about?

This project was intended to learn how kalman filters work, the idea was to mix two independent sensor data from an accelerometer and a gyroscope to remove vibrations produced when you only use accelerometer and drifting when you only use gyroscope.


The visualization is done by sending the retrieved roll, pitch and yaw angles from the sensors after applying the kalman filter to the roll and pitch angles to my laptop using Processing programming language to use its 3D visualization interface and its reference frame rotations to visualize the object orientation in other form than a tuple of 3 numbers by projecting a 3D fixed object in its original reference frame into a rotated reference frame with the corresponding RPY values.
