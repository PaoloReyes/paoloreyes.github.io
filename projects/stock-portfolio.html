What is it about?

My new project consists of a website designed to visualize the net value of my investments on a daily basis. This platform runs on a web server hosted on a Raspberry Pi 4, providing me with an accessible and autonomous solution to monitor my investments at any time. With this website, I can get a clear and up-to-date view of my investment portfolio in real-time. Leveraging the power of the Raspberry Pi 4, financial data is collected, processed, and presented in an intuitive and user-friendly interface.


Thanks to the autonomous nature of the Raspberry Pi 4, I can access the website from any device by using a service to assign a public IP, which allows me to redirect my Raspberry Pi through SSH. This provides me with flexibility and convenience to continuously track my investments, even when I'm away from home.


In addition, the website has a prediction section where I can set the desired percentage allocation for my investments. It automatically creates a plan with an expected composition portfolio and a section for upcoming investments, which tells me how to distribute the investment amount to achieve the target portfolio. This feature allows me to plan and optimize my investment strategy based on my desired allocation.
